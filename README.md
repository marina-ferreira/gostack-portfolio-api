# GoStack 12
## Challenge 02 - Node.js Fundamentals

**Express Repository API**

Create, list, edit, delete and like repositories.

```
GET /repositories
POST /repositories
PUT /repositories/:repository_id
DELETE /repositories/:repository_id
POST /repositories/:repository_id/like
```

Dependencies:

* Node - v12.17.0
* Express - v4.17.1
